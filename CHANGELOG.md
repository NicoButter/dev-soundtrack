## [0.0.1] - 2024-12-18

### Added

- Background music player with Web Audio API
- 6 music moods: Epic, Lo-Fi, Synthwave, 8-Bit, Ambient, Metal
- Sound effects system for coding events
- Sound on file save (Ctrl+S) - epic checkpoint sound
- Sound on build success - victory fanfare
- Sound on build error - dramatic sound
- Sound on git commit - achievement sound
- Sound on file open (optional)
- Sound on undo (optional)
- Beautiful webview control panel
- Volume controls for music and effects
- Mute/unmute functionality
- Keyboard shortcuts for all controls
- Configuration options via VS Code settings
- Support for custom sound files path
